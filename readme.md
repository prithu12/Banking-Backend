# BackendLedger

A basic backend ledger application.

## Features

- Record transactions
- Track balances
- Simple backend structure

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

```bash
git clone <repository-url>
cd BackendLedger
npm install
```

### Run

```bash
npm start
```
